エンジニアの転職サービスは大きく「スカウト型」と「エージェント型」に分かれます。どちらが優れているかではなく、**目的とフェーズで使い分ける**のが現実的です。本記事は両者の仕組み・向いている人・併用の考え方を、事実ベースで比較します。

## 30秒まとめ

| | スカウト型 | エージェント型 |
|---|---|---|
| 仕組み | 登録→企業/ヘッドハンターからスカウト | 登録→面談→担当者が求人提案 |
| 主な負担 | 受け身で待てる | 面談の時間が要る |
| 向くフェーズ | 情報収集・市場価値を測る | 本格的に動く・書類/面接対策 |
| 代表例 | ビズリーチ / Findy | レバテックキャリア / doda |
| 成果のハードル | 登録のみで完結しやすい | 面談実施が前提 |

## スカウト型とは

登録してプロフィール(言語・経験年数・実績)を入れておくと、企業やヘッドハンターからスカウトが届く仕組みです。**受け身で市場価値を測れる**のが利点。ビズリーチ(ハイクラス)、Findy(GitHub連携)などが代表例です。

- 向いている人: 在職中で時間が取りにくい / まず相場を知りたい / 自分から動くのが苦手
- 注意点: 良いスカウトを得るにはプロフィール記入の質が重要

## エージェント型とは

登録後にキャリアアドバイザーと面談し、非公開求人の紹介・書類添削・面接対策・年収交渉まで伴走してもらう仕組みです。エンジニア特化(レバテックキャリア等)は技術理解が深い傾向があります。

- 向いている人: 本格的に転職を進めたい / 書類・面接に不安 / 年収交渉を任せたい
- 注意点: 面談の時間が要る。複数登録すると求人が重複することがある

## どっちを選ぶ? — フェーズ別の指針

1. **まだ情報収集の段階** → スカウト型に登録して相場感をつかむ(受け身でOK)。
2. **本格的に動く段階** → エージェント型で面談し、書類・面接・年収交渉を伴走してもらう。
3. **実際は併用が一般的** → スカウトで市場を測りつつ、エージェントで深掘りする人が多いです。各社の違いは [レバテック/ビズリーチ/doda 比較](/guide/compare-levtech-bizreach-doda) を参照。

## よくある質問

**Q. 本当に無料?** はい。利用は**求職者は無料**です(採用企業が費用を負担)。
**Q. 登録するとしつこく連絡が来る?** 連絡頻度や希望は登録時/面談時に伝えれば調整できます。
**Q. 複数登録は迷惑?** 一般的な使い方です。担当に「他社も併用中」と伝えて問題ありません。

## あなたに合うサービス(無料・登録のみで始められます)

- まず市場価値を測りたい → [スカウト型: ビズリーチ](offer:bizreach)(PR/無料登録)
- 本格的に進めたい・面談で相談したい → [エージェント型: レバテックキャリア](offer:levtech)(PR/相談無料)

> 迷う場合は「スカウト型1+エージェント型1」の2社から。合わなければ退会も無料です。
